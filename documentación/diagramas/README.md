# Diagramas UML — DevNexus TFG

## Cómo renderizar los diagramas

1. Abrí https://mermaid.live
2. Pegá el contenido de cada archivo `.mmd`
3. Ajustá el tema si querés (Settings → Theme → dark/default)
4. Descargá como PNG o SVG
5. Insertá la imagen en el Google Doc donde dice `[Insertar Figura X]`
